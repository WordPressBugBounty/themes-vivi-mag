== Description ==

Vivi Mag is an elegant and stylish WordPress theme, perfect for creating online magazines, travel sites, fashion sites, or personal blogs. Vivi Mag offers an intuitive user interface and a wide range of customization options through the WordPress Live Customizer. It is also fully compatible with WooCommerce and optimized for mobile devices.

Created by ThemeinProgress, http://www.themeinprogress.com
